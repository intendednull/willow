# Routines

Prompts used in Claude web to schedule recurring agentic tasks (audits, reviews, sweeps, etc.).

Stored here for version control and documentation. The prompts run on Claude web — this directory is the source of truth for their text.

## Conventions

- One file per routine, named for what it does (e.g. `audit.md`).
- Top of each file: brief summary, expected schedule, and where it runs.
- Body: the exact prompt text, copy-pasteable into Claude web.

When updating a routine, update the file here first, then paste into Claude web.
